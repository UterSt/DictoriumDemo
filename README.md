# DictoriumDemo

Blazor WebAssembly демо-приложение для библиотеки [Dictorium](https://github.com/UterSt/Dictorium).

## О проекте

Интерактивная визуализация структур данных, реализованных в библиотеке **Dictorium**.

### Структуры данных

| Категория | Структуры |
|-----------|-----------|
| 🟢 Хэш-таблицы | Открытое хеширование, Линейное пробирование, Квадратичное пробирование, Двойное хеширование, Robin Hood |
| 🟣 Деревья (СУБД) | AVL, Красно-чёрное, B-дерево, B+-дерево |
| 🟠 Абсолют O(1) | Кукушкино хеширование, Совершенное хеширование |
| 🔵 Вероятностный поиск | Список с пропусками, Декартово дерево |

## Запуск

### Требования
- [.NET 9 SDK](https://dotnet.microsoft.com/download/dotnet/9.0)

### Команды

```bash
# Восстановить зависимости
dotnet restore

# Запустить в режиме разработки
dotnet run

# Собрать для публикации
dotnet publish -c Release -o out
```

Приложение запустится на `https://localhost:5001` (или аналогичном порту).

## Структура проекта

```
DictoriumDemo/
├── Pages/
│   ├── Home.razor          # Главная страница с обзором всех структур
│   ├── StructureView.razor # Детальный вид одной структуры
│   └── Compare.razor       # Страница сравнения структур
├── Layout/
│   └── MainLayout.razor    # Основной шаблон
├── Models/
│   └── DataStructure.cs    # Модели данных
├── Services/
│   └── DataStructureService.cs  # Данные о структурах
└── wwwroot/
    ├── index.html
    └── css/app.css          # Дизайн-система
```
